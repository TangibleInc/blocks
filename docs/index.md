## Getting started

```sh
git clone git@bitbucket.org:tangibleinc/tangible-blocks.git
cd tangible-blocks
npm install && composer install
```

## Develop

Build for development - watch files for changes and rebuild

```sh
npm run dev
```

Build for production

```sh
npm run build
```

Format to code standard

```sh
npm run format
```
